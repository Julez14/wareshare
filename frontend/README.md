# WareShare — Frontend

Frontend for the WareShare MVP: a web-based warehouse sharing marketplace where warehouse owners list space and renters book storage.

## Stack

- **Next.js** (App Router) — routing, layouts, SSR
- **React** + **TypeScript**
- **Tailwind CSS** — styling

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command            | Description               |
| ------------------ | ------------------------- |
| `npm run dev`      | Start dev server          |
| `npm run build`    | Production build          |
| `npm run start`    | Start production server   |
| `npm run lint`     | Run ESLint                |
| `npm run lint:fix` | ESLint with auto-fix      |
| `npm run format`   | Prettier format all files |
